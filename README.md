# Glow Hockey (AGameRepo)

Glow Hockey was taken from a GitHub repository called **AGameRepo**, created by one of the developers from Codeflash. The goal of the game is to score 7 pucks into the AI’s goal before the AI does. The first player to score 7 pucks into the opponent’s goal wins the game.
